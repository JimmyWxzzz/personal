This is a simple terminal simulator that only support commands that works in current directory that result in a string output.

Any input will be stored in the history window and only the result of the successful command will be stored.

To access the history of a successful command, simply double click on the command listed within the history window.

To setup the simulator:

1. Redirect yourself to the current directory

2. qmake TerminalSimulation.pro

3. make

4. ./TerminalSimulation
